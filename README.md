# HBF Run for a Reason analysis
# NOTE: This notebook is still in progress and is still being updated reguarly. 

The HBF Run for a Reason is the largest fun run in Perth. It is an annual event and was last held on 24th May 2026. There are three distances available: 3km, 12km and 21km. 

# Purpose
To investigate the results of the 2026 edition of the HBF run. 

Specific emphasis will be made to determine if the wave time cut-offs selected were appropriate and to investigate differences between runners and walkers for logistical considerations. 

# Data source
Data was scraped from the [BlueChip timing website](https://bluechipresults.com.au/results.aspx?CId=11&RId=1975). Currently, only 12km data has been scraped but the intention is that both the 3km and 21km distance will be analysed in due course.
